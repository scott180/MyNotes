1、Java基础，精通io、多线程、集合、jvm虚拟机。<br/>

   `spring`  <br/>

   `spring cloud、dubbo ； redis ; rocketMQ 、Kafka; Zookeeper ; Elasticsearch ；` <br/>
4、熟悉SQL以及性能优化，熟悉Mysql。<br/>

5、熟练使用Linux系统，熟悉Shell。<br/>
6、数据结构和算法。<br/>
   `LeetCode` <br/>
