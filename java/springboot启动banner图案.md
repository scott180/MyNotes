# springboot启动banner图案

## 配置

在Resources目录下的 application.yml 中添加以下配置

```js
spring:
  banner:
    location: /banner.txt
```

---

自定义输出banner样式
推荐生成网站
http://www.degraeve.com/img2txt.php
http://patorjk.com/software/taag/#p=display&f=Graffiti&t=java
https://www.bootschool.net/ascii-art/search
